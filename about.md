# 关于 USAIRAN

**USAIRAN** 是一个基于 GitHub Pages 的自动化新闻发布系统，专注于追踪美伊战争/中东局势的最新动态。

---

## ✨ 项目特点

- 🤖 **全自动化** - 每日自动搜索、提取、发布新闻简报
- 📰 **多源聚合** - 整合 Reuters、AP、BBC、Al Jazeera、伊朗官方等信源
- 📊 **结构化呈现** - 头条新闻 + 详细报道 + 消息来源
- 🚀 **GitHub Pages 托管** - 免费、高速、全球 CDN

---

## 📁 网站结构

```
USAIRAN/
├── _posts/           # 新闻 Markdown 文件 (YYYY-MM-DD.md)
├── _layouts/         # 页面布局模板
├── _includes/        # 导航、页眉、页脚
├── archive.md        # 历史存档页面
├── about.md          # 本页面
├── index.html        # 首页（自动从 _posts 生成）
└── _config.yml       # 站点配置
```
