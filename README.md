## 🚀 快速开始

如果你想部署自己的新闻自动化系统：

1. Fork 本仓库
2. 修改 `iran-news-publisher.yaml` 配置搜索关键词
3. 创建 GitHub Personal Access Token
4. 设置 OpenClaw 定时任务
5. 访问 `https://<your-username>.github.io/USAIRAN/`

详细文档请参考 [OpenClaw Skills 文档](https://docs.openclaw.ai)。
