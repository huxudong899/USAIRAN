ℹ️ 关于 USAIRAN

自动新闻发布系统 · AI驱动 · 开源透明

🎯 项目背景

USAIRAN 诞生于对实时国际局势追踪的需求。传统新闻媒体更新有限，人工收集信息效率低下。 于是我们开发了这个全自动化的新闻聚合系统，每天定时从全球数十家主流媒体抓取最新报道， 自动生成结构化的简报，并通过 GitHub Pages 免费托管。

🤖 核心技术：OpenClaw

OpenClaw 是一个强大的 AI Agent 框架，能够自动执行复杂的工作流：

  • 🌐 智能搜索 - 使用 Google News API、Bing News 等多引擎
  • 📄 内容提取 - 自动抓取网页正文，过滤广告和噪声
  • 🧠 AI 摘要 - 使用大语言模型生成简明摘要
  • 📝 格式转换 - 自动转换为 Markdown，保持结构清晰
  • 🚀 自动发布 - 提交到 GitHub，触发 Pages 构建

📊 数据来源

我们聚合来自 10+ 国际主流媒体 的新闻内容，包括：

国际媒体：
BBC News • Reuters • Associated Press • CNN • Al Jazeera • The New York Times

中文媒体：
新华网 • 人民网 • 央视新闻 • 财新网 • 澎湃新闻 • 中国新闻网 • 新浪军事 • 搜狐新闻

🔧 技术栈

前端Jekyll (Ruby), Liquid 模板, HTML5, CSS3
样式CSS Variables, Flexbox, Grid, Animations
部署GitHub Pages, GitHub Actions
自动化OpenClaw Agent, Cron 定时任务
媒体SVG 矢量图标, 响应式图片

🚀 自动化流程

  1. 定时触发 - 每天上午 8:30（北京时间）自动运行
  2. 新闻搜索 - 多关键词搜索："美伊战争"、"中东局势"、"伊朗"、"美国 军事"
  3. 内容抓取 - 提取前 5-10 条结果的详细内容
  4. 去重与过滤 - 去除重复和低质量内容
  5. AI 摘要 - 生成简洁的新闻摘要
  6. Markdown 生成 - 创建带 Front Matter 的日更文件
  7. Git 提交 - 自动添加到 news/ 目录并提交
  8. GitHub Pages 构建 - 触发 CI，5 分钟内全球生效
